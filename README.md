# MilMo Rebirth Client Release Distributions

This repository contains binary distributions of MilMo Rebirth Client releases.

## 🇧🇷 Como Baixar e Instalar / How to Download and Install 🇺🇸

### Português
1. Va até https://github.com/MillionMorning/Rebirth-Release/releases/latest e Clique em `MilMo-Rebirth-vx.x.0x.zip` para fazer o download do cliente do jogo.
2. Após o término do download, vá até a sua pasta de downloads e descompacte o arquivo `.zip` em uma pasta de sua preferência.
3. Inicie o arquivo `MilMo.exe` para abrir o jogo.

---

### English
1. Go to https://github.com/MillionMorning/Rebirth-Release/releases/latest and Click on `MilMo-Rebirth-vx.x.x.zip` to download the game client.
2. Once the download is complete, go to your download directory and extract the `.zip` file into a folder of your choice.
3. Run `MilMo.exe` to launch the game.
